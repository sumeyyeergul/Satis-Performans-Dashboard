# Satis-Performans-Dashboard


Power BI kullanarak hazırladığım uçtan uca satış analizi projesi.

## 🛠️ Kullanılan Teknolojiler
- Power BI (Dashboard tasarımı)
- Power Query (Veri temizleme)
- DAX (KPI hesaplamaları)

## 📊 Proje Özeti
Ham veri setlerini modelleyerek, interaktif bir satış performansı raporu oluşturdum.
- Total Sales, Total Orders gibi KPI metrikleri hesaplandı.
- Bölgesel ve kategorisel analizler için drill-down özellikleri eklendi.
